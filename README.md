# 12V AC to DC Converter PCB

This repository contains the complete design and PCB layout of a regulated 12V AC to DC power supply developed using KiCad.

## Contents
- Circuit schematic (PDF)
- PCB layout (Top & Bottom copper layers)
- Gerber files for fabrication

## Tools Used
- KiCad

## Project Status
This PCB was designed as an academic/independent project and has not yet been fabricated.
